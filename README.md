# 股票学习笔记本

个人股票学习工具，支持指标笔记、持仓管理、自选股观察和 K 线图表分析。

## 功能特性

- **指标学习笔记**：记录学习到的技术指标含义和使用方法
- **持仓管理**：管理A股持仓，记录成本价和数量
- **自选股**：添加关注股票到自选列表
- **个股详情**：查看股票 K 线、均线、成交量和 MACD/KDJ/RSI 指标
- **每日观察**：对个股添加观察记录和标签

## 技术栈

- **后端**：Flask + SQLite + baostock（A 股数据）
- **前端**：原生 HTML/CSS/JavaScript
- **图表**：TradingView lightweight-charts

## 快速开始

```bash
# 进入项目目录
cd stock-learning

# 启动服务
python3 app.py

# 访问地址
# http://localhost:5556
```

## 页面说明

| 页面 | 路径 | 说明 |
|------|------|------|
| 概览 | `/` | 持仓概览、自选股、最近观察 |
| 指标学习 | `/learning` | 技术指标笔记管理 |
| 持仓管理 | `/portfolio` | 持仓和自选股管理 |
| 个股详情 | `/stock/<code>` | K 线图表和技术指标 |

## 数据存储

- 数据库：`data/notebook.db`（SQLite）
- 无外部依赖，数据本地存储

## 股票代码

支持上交所（6开头）和深交所（0、3开头）A股股票。
